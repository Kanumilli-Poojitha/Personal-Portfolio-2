# Kanumilli Poojitha - Backend & Full Stack Developer Portfolio

Modern dark-theme personal portfolio for Kanumilli Poojitha, a backend and full stack developer focused on scalable systems, distributed architecture, and production-ready applications.

## Features

- Sticky transparent navbar with active-section highlighting
- Premium hero, about, skills, projects, contact, and footer sections
- Glassmorphism accents with smooth motion and parallax
- Accessible navigation, focus states, and reduced-motion support
- Mobile-first responsive layout tuned for deployment on Vercel
- Data-driven section content for easy portfolio updates

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Scroll
- React Scroll Parallax
- React Icons

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production Build

```bash
npm run build
```

## Deployment

1. Push the repository to GitHub.
2. Import the project into Vercel.
3. Use the default Vite build command: `npm run build`.
4. Set the output directory to `dist`.
5. Confirm the homepage metadata and social links before publishing.
